Estatística: Média, Mediana e Moda 📊

A Estatística no ENEM serve para testar a sua capacidade de interpretar tabelas, gráficos do cotidiano e calcular as chamadas "medidas de tendência central". Vamos dominar as três principais:

1. Média Aritmética

É a medida mais famosa. Para calcular, você soma todos os valores e divide pelo número total de elementos que você somou.

Exemplo Simples: As notas de um aluno foram 6, 7 e 8.
Soma = 6 + 7 + 8 = 21.
Como são 3 notas, dividimos por 3: 21 ÷ 3 = Média 7.

⚠️ Cuidado com a Média Ponderada: É quando os números têm pesos diferentes. Em vez de só somar, você multiplica cada número pelo seu peso, soma os resultados e divide pela soma dos pesos.

2. Mediana (O termo do meio)

A Mediana é o valor que fica exatamente no centro de uma lista de números. Mas atenção: para achar a mediana, existe um passo obrigatório que muita gente esquece e erra a questão:

Passo 1: Organize os números em ordem crescente (do menor para o maior). Essa lista organizada se chama Rol.
Passo 2: Encontre o centro.
• Se a quantidade de números for ÍMPAR: A mediana é o número que está bem no meio.
• Se a quantidade de números for PAR: Você pega os dois números do meio, soma eles e divide por 2.

3. Moda (O que mais se repete)

Assim como uma roupa que está na moda é aquela que todo mundo está usando, a Moda na estatística é simplesmente o valor que aparece com maior frequência (o que mais se repete) na lista.

Exemplo: Na lista [2, 3, 5, 5, 7, 8], o número 5 é a moda porque aparece duas vezes e os outros apenas uma. Se nenhum número se repetir, a lista é chamada de amodal.
