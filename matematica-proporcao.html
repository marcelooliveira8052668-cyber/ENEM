Razão, Proporção e Regra de Três 📐

Se tem um assunto que você precisa dominar para o ENEM, é este. Ele serve de base para interpretar gráficos, escalas de mapas e problemas de química.

1. O que é Razão?

Razão é simplesmente uma divisão entre duas grandezas. Ela serve para comparar dois números. A ordem importa muito!

Razão = A / B

2. O que é Proporção?

Proporção é a igualdade entre duas razões. Se duas frações são proporcionais, nós podemos usar a famosa "multiplicação cruzada" (em cruz) para descobrir um valor escondido.

3. Regra de Três Simples

É a ferramenta para descobrir um quarto valor quando já conhecemos três. Mas atenção: você precisa descobrir se as grandezas são:

  • Diretamente Proporcionais: Se uma grandeza aumenta, a outra também aumenta. (Ex: Se você compra mais pães, o preço total aumenta). Multiplica-se em cruz.
  • Inversamente Proporcionais: Se uma grandeza aumenta, a outra diminui. (Ex: Se você aumenta a velocidade do carro, o tempo de viagem diminui). Multiplica-se em linha reta.
⚠️ Como o ENEM cobra isso?
O ENEM ama misturar isso com Escala (de mapas ou maquetes). Lembre-se sempre da fórmula básica de escala:
Escala = Medida no Desenho / Medida Real
